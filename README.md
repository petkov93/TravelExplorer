# Travel Explorer 🌍✈️

A simple and responsive travel website built with HTML and CSS.

## Live Demo

https://petkov93.github.io/TravelExplorer

## Features

- Responsive design
- Hero section
- Popular destinations
- Adventure tours
- Travel tips sidebar
- Photo of the day
- Newsletter footer

## Technologies Used

- HTML5
- CSS3
- Flexbox
- Font Awesome
- Google Fonts

## Run Locally

```bash
git clone https://github.com/petkov93/TravelExplorer.git
cd TravelExplorer
python -m http.server
```

Open your browser and visit:

```text
http://localhost:8000
```

Or simply open `index.html` directly in your browser.
